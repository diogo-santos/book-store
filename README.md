# book-store

## Getting the code on your computer [Back end]
- [ ] Java 8
- [ ] Maven 3
- [ ] Import the project from GitHub https://github.com/diogo-santos/book-store

Run the app
```
cd books-api
mvn spring-boot:run
```

Execute tests
```
mvn clean package
```

## Test the App
Check API documentation at http://localhost:8080/swagger-ui.html
